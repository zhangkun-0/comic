# Comic Layout Studio

一个基于浏览器的多格漫画排版工具原型，支持通过可视化的方式设计页面布局、导入图片和对白气泡，并对画格进行图层管理与调色。

## 功能特性

- **页面与漫画区域设置**：自定义页面宽高，并指定漫画区域的上下 / 左右边距与格框粗细。页面中央会呈现 30% 透明的蓝色漫画区域，并在顶部居中显示区域尺寸提示。
- **主操作区缩放与平移**：主画布铺满编辑区域，按住左键拖动空白处即可平移，滚动鼠标滚轮可对整页进行缩放，方便调整视图。
- **格子切分与继续细分**：点击“开始切分格子”进入切分模式，在漫画区域内拖拽即可按照拖动方向预览分割，横向 / 纵向间隙可独立设定。已生成的格子仍然保持半透明蓝色并可再次切分。
- **格子形变控制**：选中格子后提供四个角节点以及四个边中节点，支持在漫画区域范围内拉伸或挪动任意边，快速生成变形格子或整体移动格子。
- **图片管理**：每个格子都可以关联本地图片，支持缩放、水平 / 垂直平移、旋转以及水平翻转；同时提供色相、饱和度、对比度调节。
- **图层面板**：独立的图层列表展示所有格子与对白框，可调整上下顺序，快速切换选中元素。
- **对白框编辑**：提供椭圆、矩形、云朵等多种对白框样式，支持文字大小、内边距设置，内容可直接在气泡中编辑。
- **即时反馈**：操作过程中配有切分预览、提示坐标和占位信息，帮助快速完成排版。

## 使用方式

1. 打开 `index.html`（可以直接在浏览器中打开文件）。
2. 在左侧面板设置页面尺寸，并根据需要调整漫画区域的上下 / 左右间隙以及格框粗细。
3. 点击“开始切分格子”进入切分模式，在蓝色漫画区域内拖动鼠标即可生成横向或纵向的分割，间隙大小由“纵切间隙 / 横切间隙”控制。分割完成后仍可继续细分任意格子。
4. 选中格子后，可通过四角与四边节点调节形状，或在左侧面板导入图片并调整各类参数。
5. 通过“新增对话框”按钮添加对白框，在气泡内直接输入文本，同时可调整样式、字号和内边距。
6. 右侧图层面板可查看所有元素并调整上下顺序。
7. 完成后可使用浏览器的“打印”或“另存为 PDF”功能导出布局。

> 目前项目为纯前端原型，不需要后端或构建步骤，所有状态均存储在浏览器内存中。
