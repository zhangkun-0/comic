# Comic Layout Studio

一个基于浏览器的多格漫画排版工具原型，支持通过可视化的方式设计页面布局、导入图片和对白气泡，并对画格进行图层管理与调色。

## 功能特性

- **页面设置**：自定义页面宽高以及横向 / 纵向间隙，画布背景会根据设定显示参考网格。
- **格子绘制与编辑**：开启绘制模式后在页面上拖拽即可生成漫画格，选中格子时会显示四个角节点，可拖动节点塑造任意四边形，或直接拖动格子整体移动。
- **图片管理**：每个格子都可以关联本地图片，支持缩放、水平 / 垂直平移、旋转以及水平翻转；同时提供色相、饱和度、对比度调节。
- **图层面板**：独立的图层列表展示所有格子与对白框，可调整上下顺序，快速切换选中元素。
- **对白框编辑**：提供椭圆、矩形、云朵等多种对白框样式，支持文字大小、内边距设置，内容可直接在气泡中编辑。
- **即时反馈**：操作过程中配有辅助轮廓、提示坐标和占位信息，帮助快速完成排版。

## 使用方式

1. 打开 `index.html`（可以直接在浏览器中打开文件）。
2. 在左侧面板设置页面尺寸与间隙，点击“开始绘制格子”进入绘制模式。
3. 在页面画布上拖动鼠标创建新的漫画格。选中格子后可以拖拽角点调节形状，或在左侧面板导入图片并调整参数。
4. 通过“新增对话框”按钮添加对白框，在气泡内直接输入文本，同时可调整样式、字号和内边距。
5. 右侧图层面板可查看所有元素并调整上下顺序。
6. 完成后可使用浏览器的“打印”或“另存为 PDF”功能导出布局。

> 目前项目为纯前端原型，不需要后端或构建步骤，所有状态均存储在浏览器内存中。
